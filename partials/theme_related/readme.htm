How to use this theme in its current form.

You can't. . . . But if you really want to, first and foremost, good luck. Here is a basic FAQ although no one asked these before.

How to create new pages
  • Go to CMS -> Pages
  • Open Home
  • Copy everything
  • Click Add
  • Add a name and Add the layout called main-layout
  • Paste everything
  • Edit ONLY where it says "Your content goes here" otherwise it will break everything and nothing will work
  • Done

DON'T USE THIS THEME IN PRODUCTION YET. LOTS AND LOTS OF THINGS ARE GOING TO CHANGE IN THE COUPLE DAYS/WEEKS.
